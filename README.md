# Тест для MTG

**Как запустить проект**

1. Клонировать репозиторий

   ```bash
   git clone https://github.com/komubosu/test-for-mtg.git
   ```

2. Установить зависимости

   ```bash
   yarn
   ```

3. Запустить виртуальный сервер

   ```bash
   yarn dev
   ```

**Что используется?**

- TypeScript
- React(class components)
- Redux
- scss
